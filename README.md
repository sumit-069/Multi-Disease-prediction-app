# 🧠 Multi-Disease Prediction System

A Machine Learning powered web application built with **Streamlit** that predicts the likelihood of multiple diseases:

- Diabetes Prediction  
- Breast Cancer Prediction  
- Heart Disease Prediction  

This project provides an easy-to-use interface where users can enter medical parameters and get instant predictions using trained ML models.

---

## 🚀 Features

- 🔍 Predicts multiple diseases in one platform  
- 📊 Uses trained Machine Learning models  
- 🖥️ Interactive UI with Streamlit  
- ⚡ Fast and real-time predictions  
- 🎯 Simple and user-friendly interface  

---

## 🛠️ Tech Stack

- **Python**
- **Streamlit**
- **Scikit-learn / XGBoost**
- **NumPy**
- **Pandas**
- **Pickle**

---

## 🤖 Models Used

- **Diabetes Prediction** → Logistic Regression / SVM / XGBoost  
- **Breast Cancer Prediction** → Logistic Regression  
- **Heart Disease Prediction** → Random Forest / Logistic Regression
